### Базовая функицональность электроплиты:

1. Класс комфорки
- Создать класс с конструктором (принимает название конфорки)
- добавить свойства: isOn (boolean), power (0-10), name (string)
Реализовать методы:
- turnOn(power) - включить конфорку на указанную мощность
- turnOff() - выключить конфорку
- setPower(newPower) - изменить мощность (с проверкой 0-10)
- getStatus() - вернуть строку со статусом ("Конфорка X: ВКЛ (5/10)")

2. Класс ElectricStove (плита)

- Создать класс плиты
- Добавить 4 конфорки (объекты Burner) с понятными именами:
frontLeft, frontRight, backLeft, backRight
- Добавить свойство isStoveOn (общее состояние плиты)
Реализовать методы:
- turnOn() - включить плиту
- turnOff() - выключить плиту (и все конфорки)
- turnOnBurner(burnerName, power) - включить конкретную конфорку
- turnOffBurner(burnerName) - выключить конкретную конфорку
- getStatus() - вывести статус всей плиты в консоль
