
## Софт для безопасной электроплиты с 4 конфорками с аналоговым управлением

### Особенности
- Обязательное включение через розетку
- Выключение плиты выключает комфорки
- Логирование статуса для проверки перегрева каждые 5 секунд
- Автовыключение и перезапуск при перегреве
- Установка мощности с помощью аналоговых ручек
- Эмуляция температуры, датчики перегрева
